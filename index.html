Prework Study Guide

HTML

  • HTML stands for HyperText Markup Language.
  • It structures the content of a webpage.
  • HTML tags define elements on the page.

CSS

  • CSS stands for Cascading Style Sheets.
  • It styles the layout of web pages.
  • CSS allows for custom colors, fonts, and layouts.

Git

  • Git is a version control system.
  • It tracks changes to files.
  • Git allows for collaboration among developers.

JavaScript

  • JavaScript makes web pages interactive.
  • It can modify HTML and CSS in real-time.
  • JavaScript is used for dynamic content updates.
